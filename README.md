# YCMAIN - 轻量工具集

你的设备管理助手，聚焦效率，让设备管理更轻松。提供课表管理、定时任务、网络控制与监控等实用功能。

---

## 📖 项目背景

这是一个采用 **Vue 3** 构建的现代化 Web 应用

---

## 🛠️ 技术栈

- Vue 3
- Vue Router
- Pinia 状态管理
- Vite

---

## 💻 运行环境与构建

### 环境要求

- Node.js 24.13.0

### 项目搭建

```bash
# 安装依赖
npm install
```

### 开发运行

```bash
# 启动开发服务器（支持热重载）
npm run dev
# 启动文档服务器（支持热重载）
npm run docs:dev
# 注：两个服务器需分别启动，dev先启动，docs:dev后启动
# 再注：文件路径可以有中文，但不能有特殊符号，比如说“（”，否则会无法访问
```

### 生产构建

```bash
# 类型检查、编译并压缩为生产环境代码
npm run build
```

### 代码检查

```bash
# 使用 ESLint 进行代码规范检查
npm run lint
```
